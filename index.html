Windows apps will run on Chromebook too

Thanks to Crossover, after Google play, windows applications can be utilized on Chromebook's devices. In the following video it is shown a game (Limbo) for Windows running on a Chromebook. The devices must have a x86 processor and not an ARM.

Le app di windows gireranno anche su Chromebook

Grazie a CrossOver, dopo Google play, anche i programmi di Windows potranno essere utilizzati sui dispositivi Chromebook. Nel seguente video viene mostrato un gioco (Limbo) per Windows che gira su un Chromebook. Idispositivi dovranno avere, però, un processore x86 e non un processore ARM.
